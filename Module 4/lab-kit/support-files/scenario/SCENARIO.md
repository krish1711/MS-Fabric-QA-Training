# Scenario

Contoso Outdoor Retail uses Microsoft Fabric to produce daily regional sales reporting for multiple customer groups.

The QA team must validate four Day 4 concerns:

1. The pipeline runs activities in the correct sequence.
2. Failures are clearly visible in run history and can be retested after a fix.
3. The semantic model exposes correct business logic for reporting.
4. Regional users are restricted to their own data through row-level security.

The Day 4 lab uses a small but realistic dataset so learners can prove those behaviors end to end.
