# Scenario

Contoso Outdoor Retail is preparing a Microsoft Fabric solution for release.

The solution already produces the right business results, but QA now needs runtime and readiness evidence. The Day 5 lab focuses on these questions:

1. Can QA distinguish fast and slow activities from run history?
2. Can QA explain a failure pattern and a successful rerun?
3. Can QA validate the final reporting outputs against the truth set?
4. Can QA automate the stable checks using pytest without trying to automate everything?
