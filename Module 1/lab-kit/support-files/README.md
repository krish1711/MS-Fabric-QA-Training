# Support Files

This folder contains all support files needed for the Day 1 lab.

## Folder Map

- `data/`: CSV data for the Lakehouse
- `notebooks/`: notebook code used to create Lakehouse tables
- `sql/`: Warehouse setup scripts and validation queries
- `scenario/`: business scenario, data dictionary, and expected results
- `templates/`: learner worksheet and evidence log template

## How These Files Are Used

- Upload the CSV files from `data/` into the Lakehouse
- Paste the notebook script from `notebooks/` into a Fabric notebook attached to the Lakehouse
- Copy the SQL scripts from `sql/` into the Warehouse SQL editor
- Use the documents in `scenario/` to explain the lab context and verify expected results
- Use the files in `templates/` as learner handouts
