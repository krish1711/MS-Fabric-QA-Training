# Day 1 Lab Kit

This folder contains the complete lab package for **Module 1: Fabric Basics for QA**.

## Use This Folder In This Order

1. `01-INSTRUCTOR-SETUP.md`
2. `02-LEARNER-LAB-GUIDE.md`
3. `support-files/`

## What This Lab Kit Includes

- A step-by-step instructor setup guide
- A full end-to-end learner lab guide written in an exercise-and-task format
- CSV support data for the demo Lakehouse
- A notebook script to create Lakehouse tables
- SQL scripts to create and load Warehouse tables
- Validation queries
- A learner worksheet and evidence log template
- Scenario, data dictionary, and expected results documents

## Recommended Demo Asset Names

Use these names unless your environment already has a preferred naming standard:

- Workspace: `ws_day1_contoso_outdoor`
- Lakehouse: `lh_day1_contoso`
- Warehouse: `wh_day1_contoso`
- Pipeline: `pl_day1_navigation_demo`
- Notebook: `nb_day1_load_lakehouse_tables`
- Report: `rpt_day1_order_status`

## Lab Objective

By the end of the lab, learners should be able to:

- Locate the major Microsoft Fabric items in a workspace
- Explain the difference between Lakehouse and Warehouse in QA terms
- Identify where QA evidence can be found
- Describe how workspace-based customer isolation should be validated

## Folder Structure

- `support-files/data/`: CSV files to upload to the Lakehouse
- `support-files/notebooks/`: notebook code to create Lakehouse tables
- `support-files/sql/`: SQL scripts for Warehouse setup and validation
- `support-files/scenario/`: scenario, data dictionary, and expected results
- `support-files/templates/`: worksheet and evidence log templates
